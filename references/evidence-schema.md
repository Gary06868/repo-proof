# Evidence Schema

RepoProof writes:

- `repoproof-report.json`: machine-readable evidence.
- `repoproof-report.md`: human-readable summary.
- `repoproof-fixes.patch`: optional patch from fix mode.
- `repoproof-action.yml`: optional GitHub Action companion.

Reports include:

- schema version
- mode and status
- repository path and commit
- project type and package manager
- extracted README steps
- safety model
- cleanroom provenance
- issues
- executed command evidence
- fix metadata
